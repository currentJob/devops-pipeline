## 🌅 아침 — 냉두부 된장국 + 보리밥

### 재료 (1인분)
| 재료 | 분량 | 비고 |
|------|------|------|
| 보리쌀 | 1컵 (180 ml) | 전날 밤 불리면 ↑ |
| 연두부 | 1/2모 (150 g) | 냉장 보관 상태로 사용 |
| 애호박 | 1/4개 | 반달 썰기 |
| 된장 | 1 큰술 | |
| 다시마 육수 | 300 ml | 멸치 육수 대체 가능 |
| 대파 | 약간 | 어슷 썰기 |
| 참기름 | 1/2 작은술 | 마무리용 |

### 조리 순서 (약 20분)
1. **보리밥** — 보리쌀을 씻어 물 1.2컵과 함께 취사 버튼 누르기.
2. **육수 끓이기** — 냄비에 다시마 육수를 붓고 중불 가열.
3. **된장 풀기** — 육수가 따뜻해지면 된장을 체에 거르며 풀어 넣기.
4. **채소 넣기** — 애호박 → 2분 끓인 뒤 연두부를 숟가락으로 크게 떠넣기.
5. **마무리** — 대파 넣고 불 끄기 → 참기름 두르기.

> [!tip] 포인트
> 여름엔 된장국을 **살짝 싱겁게** 끓이고, 냉두부를 넣으면 국물이 시원하고 소화가 편합니다.

## ☀️ 점심 — 닭가슴살 냉채 비빔국수

### 재료 (1인분)
| 재료 | 분량 | 비고 |
|------|------|------|
| 소면 | 100 g | |
| 닭가슴살 | 100 g | 삶아서 결대로 찢기 |
| 오이 | 1/2개 | 채 썰기 |
| 당근 | 1/4개 | 채 썰기 |
| 깻잎 | 4장 | 가늘게 채 썰기 |
| **[양념장]** | | |
| 고추장 | 1 큰술 | |
| 식초 | 1 큰술 | |
| 설탕 | 1/2 큰술 | |
| 간장 | 1/2 큰술 | |
| 참기름 | 1 작은술 | |
| 다진 마늘 | 1/2 작은술 | |
| 통깨 | 약간 | |

### 조리 순서 (약 20분)
1. **닭 삶기** — 닭가슴살에 소금·생강 약간 넣고 12분 삶기 → 식혀서 손으로 결대로 찢기.
2. **양념장 섞기** — 위 양념 재료를 볼에 모두 넣고 잘 섞어두기.
3. **소면 삶기** — 끓는 물에 소면 2분 30초 삶기 → 찬물에 헹궈 물기 빼기.
4. **담기** — 소면에 닭가슴살·오이·당근·깻잎 올리기 → 양념장 뿌리기 → 통깨 마무리.

> [!tip] 포인트
> 국수를 얼음물에 담갔다 헹구면 면발이 탱탱해집니다. 양념장은 먹기 직전에 올려야 면이 불지 않습니다.

## 🌙 저녁 — 삼치 소금구이 + 가지나물 + 흰밥

### 재료 (1인분)
| 재료 | 분량 | 비고 |
|------|------|------|
| 삼치 | 1토막 (150 g) | 6월이 제철 |
| 소금 | 약간 | |
| 식용유 | 1 큰술 | |
| **[가지나물]** | | |
| 가지 | 1개 | |
| 간장 | 1 큰술 | |
| 참기름 | 1 작은술 | |
| 다진 마늘 | 1/2 작은술 | |
| 통깨 | 약간 | |

### 조리 순서 (약 20분)
1. **삼치 밑간** — 삼치에 소금을 앞뒤로 고르게 뿌리고 10분 두기.
2. **가지 찌기** — 가지를 반으로 가른 뒤 전자레인지 3분 → 식혀서 손으로 세로 방향으로 쭉쭉 찢기.
3. **가지나물 무치기** — 찢은 가지에 간장·참기름·다진 마늘·통깨를 넣고 무치기.
4. **삼치 굽기** — 달군 팬에 식용유 두르고 중불 → 삼치 앞면 4분, 뒤집어 3분 굽기.
5. **담기** — 흰밥 + 삼치 소금구이 + 가지나물 차려내기.

> [!tip] 포인트
> 삼치는 뒤집기 전 **건들지 않아야** 껍질이 안 달라붙습니다. 뚜껑을 닫고 속까지 익히세요.

> [!warning] 주의
> 삼치는 가시가 촘촘합니다. 어린이·어르신 식사 시 **가시 제거**를 반드시 확인하세요.
